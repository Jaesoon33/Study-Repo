1. CSS Grid 란
    1) 2차원 레이아웃 시스템(행&열 동시 제어 가능)
    2) Flexbox: 1차원(한 방향) vs Grid: 2차원(양 방향)
    3) complex layout을 간단한 코드로 구현 가능

2. Grid Container와 Grid Item
    1) Grid Container (display: Grid)
        - 컨테이너를 Grid로 선언
    2) 직계 자식 요소들이 자동으로 Grid Item이 됨

3. CSS Grid Properties
    1) Grid Container Properties
        - 부모 요소(Container)에 적용되는 속성
        - 그리드의 전체 구조, 행&열 정의, 간격 설정
        - 콘텐츠/아이템 정렬 등을 담당
        - 그리드 구조 정의
            - display: // 그리드 컨테이너를 선언
            - grid-template-columns: // 명시적의 열(Track)의 크기를 정의
            - grid-template-rows: // 명시적의 행(Track)의 크기를 정의
            - grid-auto-flow: 자동 배치 알고리즘 방식을 정의 (empty area를 자동으로 채우는 알고리즘) 
            - grid-template-areas: 영역(Area) 이름을 참조하여 template을 생성
            
    2) Grid Item Properties
        - 자식 요소(Item)에 적용되는 속성
        - 개별 아이템의 위치, 크기(span), 정렬 
        - 배열 순서, 적층 순서 등을 담당
        - grid-column: 열의 시작/끝 위치(column number)
            - grid-column-start: 그리드 아이템의 행 시작 위치 지정
            - grid-column-end: 그리드 아이템의 행 종료 위치 지정
            - grid-area: 영역 이름 설정 또는 grid-row/column 단축 속성으로 사용

4. display 
    1) 그리드를 사용하기 위해 컨테이너에 필수로 정의함
        - 정의된 컨테이너의 자식 요소들은 자동으로 Grid Item으로 정의됨
    2) 값
        - grid : Block 특성의 Grid Container를 정의             // one line one element
            - 다중 Container 2개로 확장
        - inline-grid : Inline 특성의 Grid Container를 정의     // one line many element
