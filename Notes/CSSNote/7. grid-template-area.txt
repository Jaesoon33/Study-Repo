1. grid-template-areas
    1) 지정된 그리드 영역 이름(grid-area)을 참조해 그리드 템플릿을 생성함.
        - 영역 이름으로 레이아웃을 정의
        - 각 문자열 = 하나의 행(row)
        - 같은 이름 반복 = 영역 확장(span)
    2) grid-area는 Grid Item에 적용하는 속성임.

2. 빈 영역 표기법
    1) .(마침표) = 빈 셀 (empty cell)
    2) none = 빈 셀

3. grid-template 단축 속성
    1) areas + rows + columns를 한번에 정의하는 syntex
        - 코드 간결화, 실무 추천, 가독성&직관성 향상

    2) 문법 구조
        
        grid-template: "area names"         행높이  <--- 1행
                       "area names"         행높이  <--- 2행
                       "area names"         행높이  <--- 3행
                       / 열너비들

    3) 단축 속성의 장점
        - 레이아웃구조가 한눈에 보임
        - 사용되는 코드의 양이 줄어듦 (3속성 -> 1속성)
        - 영역과 크기가 같은 줄에 있어서 직관적
        - 디자이너와 소통할 때 설명하기 쉬움
