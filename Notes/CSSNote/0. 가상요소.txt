1. position: relative 와 absolute 관계
    1) 주요개념
        - position: absolute는 가장 가까운 position된 부모 요소를 기준으로 위치가 설정됨
        - 부모에 position: relative가 없으면 --> view port(device display) 기준으로 이동됨
        - 부모에 position: relative가 있으면 --> 부모 요소 기준으로 위치가 지정됨

2. ::after 가상 요소 
    1) 가상 요소 (pseudo-element))는 HTML에 직접 작성하지 않아도
        CSS로 생성할 수 '있는 가상의 요소'
        - ::before      요소의 내용 '앞'에 가상 요소 삽입됨
        - ::after       요소의 내용 '뒤'에 가상 요소 삽입됨

    2) CSS 가상 요소와 position 정리
        - ::after 가상 요소
            - CSS로 HTML 없이 요소의 컨텐츠 뒤에 가상 콘텐츠를 생성
            - content 속성이 필수  (없으면 랜더링 안됨)
            - DOM 존재하지 않음
            - 장점
                - HTML이 깔끔
                - CSS 일괄 관리 가능
                - 그라데이션 + 애니메이션 효과 적용 가능
        
        - ::before 가상 요소
            - CSS로 HTML 없이 요소의 컨텐츠 앞에 가살 콘텐츠를 생성
            - ::after와 동일하게 content 속성 필요

        - ::after vs ::before 차이
        가상요소    위치    사용 예시   
        ---------------------------------------
        ::before 컨텐츠 앞 리스트 아이콘, 인용부호
        ""after  컨텐츠 뒤 밑줄, 장식 요소를

        - position 주요 개념
            - position: absolute는 가장 가까운 position된 상위 요소를 기준으로 위치가 정해짐
            - 부모에 position: relative가 없으면 --> 뷰포트(화면) 기준으로 이동
            - 부모에 position: relative가 없으면 --> 부모 기준으로 위치 지정
