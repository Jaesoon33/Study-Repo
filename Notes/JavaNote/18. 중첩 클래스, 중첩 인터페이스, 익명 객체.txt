1. 중첩 클래스 (내부 클래스 inner class)
	1) 클래스 안에 만든 클래스
	2) 서로 관련 있는 클래스를 하나로 묶을 떄 사용함
	3) ex
		class Out{
			class In{
			
			}
		}
	4) 종류
		- 인스턴스 내부 클래스
		- 정적 내부 클래스
		- 지역 내부 클래스
		- 익명 내부 클래스

3. 익명 내부 클래스
	1_ 이름이 없는 클래스    
	2) 클래스 정의와 객체 생성을 동시에 수행 가능
	3) 주로 인터페이스나 추상클래스를 일회성으로 구현할 때 사용함
	4) 코드의 간결성과 캡슐화를 제공하지만 재사용성은 낮음
