1. 함수(function)
	1) 함수 정의
	2) 함수 호출(사용)
	
2. 메서드(Method)
	1) 자바에서는 함수를 메서드라 함
	2)크게 메서드 선언과 메서드 본문으로 나눌 수 있음
		- 메서드 선언 (Method Declaration)
			- public static void arraycopy(Object src,
											int srcPos,
											Object dest,
											int destPos,
											int length)
				- public 
					- 다른 클래스에서 호출할 수 있는 메서드라는 의미
				- static 
					- 객체를 생성하지 않고 호출할 수 있는 정적 메서드
		- 메서드 본문 (Method Body)
			- 메서드가 수행해야 되는 코드 블록
			- 메서드를 호출하면 메서드 본문이 순서대로 실행됨
				- 메서드를 호출하는 곳에서는 메서드 선언은 알지만 본문은 몰라도 됨(don`t care)
			- 메서드의 실행 결과를 반환하려면 return문 사용해야 함
				- return문 다음에 반환할 결과를 적어주면 됨

3. 메서드 오버로딩(Overloading)
	1) 같은 이름의 메서드의 매개변수를 다르게 하여 매개변수를 여러 개 정의하는 것
	2) 조건
		- 메서드 이름이 같아야함
		- 매개변수의 타입, 개수, 순서 중 하나 이상이 달라야함
	3) 성립이 안되는 조건
		- 반환 타입만 다른 것은 오버로딩이 아님
		- 매개변수의 이름만 다른 것은 오버로딩이 아님
	
4. 가변 인자(Varargs)
	1) 문법
		- int... numbers(var)
	2) 의미	
		- 0개 이상의 int를 받아 내부적으로 int[] 배열을 처리
	3) 많은 Java  API가 가변 인자를 활용함
