1. web의 생성 배경 (Browser)
    1) HTML, HTTP, URL -- 팀 버너스 리 / 1989
    2) Mosaic Browser / 1993
    3) Netscape Navigator / 1995 브라우저 점유율 75%
        - 브라우저는 준비됐으나, 사이트가 별로 없음
        - Java Applet

2. JS의 생성과 발전
    1) 웹사이트를 쉽게 구동시키기 위한 환경을 Netscape에 Embed하자
    2) Brendan Eich가 Netscape에 영입됨
        - LiveScript 완성 / 1995
        - JavaScript로 rename
    3) TC39 위원회
        - ECMA에 찾아가 표준으로 제안함
    4) 버전
        - 1995 LiveScript -> JavaScript
        - 1997 ECMAScript 1
        - 2015 ECMAScript 6 (ES6 = ES2015) - 주요 기능 추가
        - 2016 ECMAScript 7
        - ECMAScript 2017... 

3. JavaScript Runtime
    1) JavaScript 코드가 실행되는 환경
        - JS engine (cooker) 
            - 메모리 관리
                - 변수, 객체 저장소.. 
            - 콜스택
                - 함수 실행 
                - 순서 관리
        - APIs (tool)         
            - 환경이 제공하는 기능
                - DOM(Document Object Model): 화면 조작
                - fetch(): 서버 통신
                - console.log() etc..
        - 대기열 (order)
            - Callback Queue
        - 이벤트 루프 (clerk: call order)
