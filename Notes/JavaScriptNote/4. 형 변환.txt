1. 형 변환
    1) Java, C, C++ 언어는 변수 선언 시 자료형을 명시
        - 해당 자료형에 알맞는 값만 변수에 할당할 수 있음
    2) JS는 변수 선언 시에 변수의 자료형을 미리 선언하지 않음
        - JS는 프로그램 실행 중(Runtime)에 자료형이 변환되는 언어이기 때문임
    3) JS는 자료형이 유연하게 변환되는 동적 타입 언어임
        - 특정 연산을 위해 자료형이 변환되는 경우도 있음

2. 묵시적 형 변환
    1) 자료형이 자동으로 변환되는 형 변환을 말함
    2) js 자동
    3) *, /, - : 문자열 -> 숫자
        + (문자열 연결: strcat) : 숫자 -> 문자열

3. 명시적 형 변환
    1) 직접 의도적으로 자료형을 변환시키는 것을 말함
    2) 개발자 직접
    3) 문자열 -> 숫자 : parsing function (wrapper class) 사용 
       숫자 -> 문자열 : String(), .toString(), + ''
